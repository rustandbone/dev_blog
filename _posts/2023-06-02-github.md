### git 명령어

```
git clone 링크

git remote -v

git add .

git commit -m '커밋 메시지 입력'
git commit

git push origin training

git switch origin

cd ..

git status

git log --oneline

git init

git remote -v

git remote add upstream ''
git branch -M main
git push -u upstream main
```
