# 테킷 멋쟁이사자처럼 JavaScript

## [Node: nodeType property][]

- Node.ELEMENT_NODE (1)
  - An Element node like <p> or <div>.
- Node.ATTRIBUTE_NODE (2)
  - An Attribute of an Element.
- Node.TEXT_NODE (3)
  - The actual Text inside an Element or Attr.
- Node.COMMENT_NODE (8)
  - A Comment node, such as <!-- … -->.
